# Repository Instructions

- Write all code, code comments, documentation, commit messages, and project
  configuration text in English.
- Conversation with contributors may happen in any language.
- External examples may preserve their source language when they represent real
  incoming data, such as bank SMS parser fixtures.
- When using Render MCP for this repository, if Render MCP reports that no
  workspace is selected, list workspaces first so the MCP server can auto-select
  the single available workspace.
